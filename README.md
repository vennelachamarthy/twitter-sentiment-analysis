Twitter Sentiment Analysis


NLP project classifying 1.6M tweets as positive or negative using 

Machine Learning and Deep Learning.


Results:

Model: Logistic Regression 
Accuracy(AUC): 77.84% (\~0.86)

BERT (fine-tuned) : In progress 



Project Structure

1. `notebooks/` — full analysis with visualizations

2\. `models/` — saved trained models

3\. `data/` — predictions export CSV

4\. `app/` — Streamlit web app (coming soon)



Tech Stack

Python | scikit-learn | HuggingFace Transformers | NLTK |
matplotlib |seaborn | Streamlit
